The MUYA Transcription Editor

Version

This is an electron application which provides a locally running version of the Online Transcription Editor (OTE) also developed as part of the MUYA project by Center for Digital Humanities, University of Trier.

To find out more about the editor itself select 'About OTE' from the menu.

This application was written as part of the ERC funded MUYA project hosted at SOAS.

Technical development was by ITSEE, University of Birmingham.
